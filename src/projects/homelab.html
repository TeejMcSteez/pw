Personal Homelab

Services

  • Home Assistant
  • TrueNAS Scale
  • NGINX Reverse Proxy + Web Server
  • N8N
  • AdGuard Home
  • Frigate
  • Jellyfin
  • GoUp

All of this runs on a single node Proxmox server with most of the services being docker images running on a Debian VM or directly on Debian host
Services that are images themselves like Truenas Scale and Home Assistant OS are mounted directly to a Proxmox VM disk
Backups happen everynight to an offsite (grandpa's house) server which runs Proxmox Backup Server connected via Tailscale controlled with my Headscale control plane
Their is an issue such that if Headscale is not up and running I cannot access the server but for security reasons I am okay with this
