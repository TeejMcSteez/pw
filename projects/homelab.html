Personal Homelab

Services

  • Home Assistant
  • TrueNAS Scale
  • NGINX Reverse Proxy + Web Server
  • Gitea
  • N8N
  • AdGuard Home
  • Other Auxillary services like loggers

All of this runs on a single node Proxmox server with most of the services being docker images and the ones that aren't run directly on the virtual machine which are all debian besides TrueNAS Scale and Home Assistant which are their own respective ISO's or images. Backups happen every 24 hours at 3am synced first to an external USB-C SSD and then to an internal SATA HDD with rsync and cron.
